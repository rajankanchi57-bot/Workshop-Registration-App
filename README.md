# Workshop Registration & Confirmation App

## Description

The Workshop Registration & Confirmation App is a React-based web application developed to manage workshop registrations efficiently. Users can register by entering their name, email, and workshop details. The application displays a confirmation message after successful registration and prevents duplicate entries using email validation.

## Features

* Registration Form
* Participant List
* Confirmation Message
* Duplicate Entry Prevention
* User-Friendly Interface

## Technologies Used

* React JS
* CSS
* Vite
* Node.js
